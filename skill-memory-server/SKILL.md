# memory-server

O-Mind 自建记忆服务，通过 HTTP API 调用本地 memory-server。

## Tools

### save_memory
保存重要记忆到本地服务器。

### search_memories
搜索相关记忆。

### get_memories
获取所有记忆列表。

### get_stats
获取记忆统计信息。
